# 二次函数

**阶段:** [[初中数学]] · 数与代数
**前置知识:** [[一次函数]] · [[一元二次方程]]
**之后会用到:** [[函数进阶]] · [[导数]]

## 是什么
y = ax² + bx + c,图像是**抛物线**(开口的曲线)。

## 核心要点
- a 决定开口方向(a>0 朝上,a<0 朝下)和胖瘦
- 顶点:最高 / 最低点
- 对称轴:x = −b/2a
- 与 [[一元二次方程]] 的联系:图像与 x 轴交点 = 方程的解

## 为什么重要
第一个"非直线"函数,理解曲线、最值。导数最早就用来求这类曲线的斜率。
