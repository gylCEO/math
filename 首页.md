## 怎么看关系图谱

打开 Obsidian → 左侧边栏点「关系图谱 / Graph View」图标 → 看到整张知识网络。
每篇笔记顶部都标了「前置知识」和「之后会用到」,所以图谱里能看清依赖脉络。

## 跨设备同步备忘

换新设备:
1. 装 Obsidian + git
2. `git clone https://github.com/gylCEO/math.git`
3. Obsidian「打开文件夹作为仓库」,选 clone 下来的文件夹

日常:
- 开始学之前:`git pull`
- 学完记完:`git add -A && git commit -m "更新笔记" && git push`
