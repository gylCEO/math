# 英语名词与冠词

**阶段:** [[英语基础]]
**前置:** [[英语核心词汇]]
**之后会用到:** [[英语基础句型]]

## 名词单复数
- 大多数加 -s:book → book**s**
- 特殊:child → children, man → men

## 冠词 a / an / the
- **a / an**:泛指"一个"(an 用在元音音开头前:an apple)
- **the**:特指"那个"(双方都知道的)

## 例子
- I have **a** dog. 我有一只狗。
- **The** dog is cute. 那只狗很可爱。

## 提示
中文没有冠词,这是中国人最容易漏的地方,多注意。
